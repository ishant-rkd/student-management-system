# Student Management System

## Description
A full-stack Student Management System built using Spring Boot and MySQL with a simple HTML, CSS, and JavaScript frontend.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- REST APIs
- HTML, CSS, JavaScript

## Features
- Add student
- View students
- Delete student
- MySQL database integration

## How to Run
1. Clone the repository
2. Configure MySQL database
3. Run Spring Boot application
4. Open http://localhost:8080
